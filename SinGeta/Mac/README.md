# Mac を 新下駄配列 にするスクプリト

* SinGeta_Layout.json
JISキーボードにもUSキーボードにも対応しています。
Karabiner-Elements を動かしたまま、システム環境設定 → キーボード → キーボードの種類を変更　からお使いのキーボードの種類を設定してください。

## 新下駄配列 +ローマ字入力

入力方法にローマ字入力を使うように、ＩＭを設定します。
Karabiner-Elements で、英語系の配列より上に登録してください。
